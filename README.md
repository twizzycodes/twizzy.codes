# twizzy.gg

Personal portfolio site. Live at [twizzygg.vercel.app](https://twizzy.codes).

![Vercel](https://img.shields.io/badge/Vercel-000000?style=flat-square&logo=vercel&logoColor=white)

---

## running locally

```bash
git clone https://github.com/bigcheesh/twizzy.gg
cd twizzy.gg/twizzy.gg
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## deploying

Deployed automatically via Vercel on push to `main`. To deploy manually:

```bash
npm run build
vercel --prod
```
